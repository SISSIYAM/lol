# YouTe

城市共享停车桩，完善支持自行车和汽车。基于vue 2.0 + 混合开发的移动端App。

### 技术架构

- 编译打包工具

  webpack@4.8.3

- 包管理工具

  npm@5.6.0

- 服务端数据请求

  axios@0.18.0

- 视图组件

  vue@2.5.2

- 状态管理

  vuex@3.0.1

- 前端UI库

  element-ui@2.4.1
